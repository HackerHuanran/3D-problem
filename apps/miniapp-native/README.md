# 微信原生小程序工程

这个目录现在已经可以作为“微信开发者工具直接打开”的项目来使用。

目录：

- `apps/miniapp-native`

当前已接入：

- 真实问题库：`problems`
- 微信身份自动建档：`profiles`
- 收藏：`problem_favorites`
- 浏览历史：`problem_history`
- 云函数：`cloudfunctions/miniappAuth`

## 第一步：先部署云函数

你需要先把这个云函数部署到你当前 CloudBase 环境：

- `cloudfunctions/miniappAuth`

在微信开发者工具里可以直接上传部署，或者在云开发控制台部署都可以。

## 第二步：用微信开发者工具打开

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择：

```txt
/Users/tree/3D-problem/apps/miniapp-native
```

4. `AppID`：
   - 如果你已经有小程序 AppID，就填你自己的
   - 如果还没有，可以先用“测试号 / 游客模式”看界面

## 第三步：开通云开发

在微信开发者工具里确认：

1. 打开“云开发”面板
2. 环境 ID 选择：

```txt
problem-d1gg06meg3dd7da6b
```

3. 确认云数据库里这些集合存在：
   - `problems`
   - `profiles`
   - `problem_favorites`
   - `problem_history`

## 现在能看到什么

1. 首页：
   - 搜索问题
   - 查看问题列表
   - 最近浏览
   - 进入账号中心

2. 初筛页：
   - 按阶段 / 机型 / 材料筛选候选

3. 详情页：
   - 查看原因
   - 查看解决步骤
   - 收藏
   - 自动记录浏览历史

4. 账号页：
   - 自动使用微信身份建档
   - 查看收藏数量
   - 查看浏览数量

## 说明

这版为了让你“直接在微信开发者工具里打开”，我采用了微信原生小程序工程，而不是继续卡在 `uni-app` 编译链上。

后面如果你要：

- 拍照识别
- 图片上传
- 审核流
- 更复杂的个人中心

都可以继续在这个原生小程序工程上往下做。
